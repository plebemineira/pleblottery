# pleblottery

![](dwarf.png)

A [SV2/SRI](https://stratumprotocol.org)-based hashrate aggregator for a pleb-friendly solo/lottery Bitcoin mining experience.

We leverage:
- SV2 Mining Proxy
- SV2 Template Provider

to allow plebs to aggregate the hashrate of their multiple ASIC devices into a single solo mining "lottery ticket".
