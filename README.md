# pleblottery

![](dwarf.png)

A [StratumV2 Reference Implementation](https://stratumprotocol.org)-based hashrate aggregator for a pleb-friendly solo/lottery Bitcoin mining experience.

We leverage:
- SV1/SV2 Mining Proxy
- SV2 Template Provider

to allow plebs to aggregate the hashrate of their multiple mining devices into a single solo mining "lottery ticket".
